# PassForge

PassForge is a single-file static password generator web app that creates cryptographically secure passwords using the Web Crypto API (`crypto.getRandomValues()`). This repository contains the full UI, styles, and JavaScript in `index.html`.

## Features

- Secure random generation via `crypto.getRandomValues()`
- Single dark theme with gold accents
- Password visibility toggle, copy, and download
- Entropy evaluation and multiple password generation
